# STALKER-ANOMALY-BAHASA-INDONESIA
Proyek ini bertujuan untuk menyediakan lokalisasi penuh Bahasa Indonesia untuk mod standalone S.T.A.L.K.E.R. Anomaly. Fokus utama kami adalah menerjemahkan dialog, antarmuka pengguna (UI), deskripsi item, dan panduan PDA agar pemain Indonesia dapat menikmati pengalaman di The Zone dengan lebih mendalam.
